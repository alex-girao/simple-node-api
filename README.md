# simple-node-api
Simple Node API Example

## Features

  * List Artists
  * List Artist
  * Create Artist
  * Update Artist
  * Remove Artist

## Quick Start

To start the application is very simple.

  Install dependencies by Node:
```bash
$ npm install
```
  or Install dependencies by Yarn:
```bash
$ yarn
```

  Start the server:

```bash
$ node index.js
```
